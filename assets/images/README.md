# Assets - Gambar Profil

Folder ini digunakan untuk menyimpan gambar/aset yang digunakan dalam aplikasi MyPorto.

## Instruksi Penambahan Foto Profil

1. Letakkan foto profil Anda di folder ini dengan nama: `profile.png`
2. Ukuran gambar yang direkomendasikan: 500x500 px
3. Format yang didukung: PNG, JPG, JPEG
4. Aplikasi akan secara otomatis menampilkan foto profil dari folder ini

Contoh struktur:
```
assets/
  images/
    profile.png   <-- Letakkan foto profil Anda di sini
    README.md     <-- File ini
```

Setelah menambahkan foto, jalankan `flutter clean` kemudian `flutter run` untuk refresh aset.
